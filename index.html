The World of Elements

  • High Concept: Go around the world and learn about the elements!
  • Player Experience Goals: A way to have fun while learning about the periodic table of elements and some basic chemistry!
  • Genre: This is a casual adventure game!
  • Game play: The player has to bump into the elements to pick them up. Some elements include Earth:
    And Diamonds are intended to be special pick ups:
  • Prototype goal: To get a working game showing properly! Plus movement and sound...
  • Features: The player is able to scroll around the world gathering elements to learn about them!
  • Setting/Premise: You are a scientist who happens to have access to a tank! You have to drive around gathering elements in order to use them for experiments and keep the world from collapsing!
  • Story: The elements have gone crazy and you have to go collect the ones running wild to bring peaceback to the Earth!
  • Target Audience: I'm trying to reach any and all people at least old enough to be in elementary school.
  • Play time: This only takes a couple minutes to read the instructions and play the game (for now).
  • Strategies: What player strategies do you expect will be effective at playing this game?
  • Prototype Assets: I used the Phaser example game "Tanks" as a basis for the game to learn about the movement, plus the diamond image from the basic phaser tutorial.
  • Prototype Closest Other Game: The Phaser prototype "Tanks" is the most similar since I've used that as a basis to start from.
